*Pluralisation with simple backend*

To get rails to pluralise all languages you needset up pluralisation rules
Copy https://github.com/svenfuchs/i18n/blob/master/test/test_data/locales/plurals.rb to config/locales/plurals.rb

Ensure that your app reads ruby files for localisation
In config/application.rb
config.i18n.load_path += Dir[Rails.root.join('my', 'locales', '*.{rb,yml}').to_s]

In config/initializers make pluralization.rb
require "i18n/backend/pluralization"
I18n::Backend::Simple.send(:include, I18n::Backend::Pluralization)

Languages with tricky plurals like cs, ja etc should now work
Languages that have the default pluralisation, same as en, :one, :other do not need the above.
